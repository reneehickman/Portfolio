About Me

Since I was young, I have always been inspired by the world around me. For the past 8 years, video editing and digital design has been the main way I express my love for the things I'm passionate about. Looking at my past work, I see that I'm ultimately driven by my unique perspective on relating to the world around me. No matter the medium, I enjoy inspiring the creativity in others.

Portfolio

Tampa Bay Lightning Hangman

A hangman style game with a Tampa Bay Lightning twist created using vanilla JavaScript.

Star Sign Trivia

Test your knowledge on Astrology! Star Sign Trivia is an application that uses JavaScript for the logic and jQuery to manipulate HTML.

Bamazon (node-mysql)

Bamazon is an Amazon-like storefront CLI application created using Node.js, MySQL & inquirer npm. Output is styled with Chalk and tty-table NPM's. This app takes in customer orders, calculates the sales price, and depletes stock from the store's Inventory.

TaskBidder

TaskBidder is a task management application created using React and Material-UI for a group project. This application allows users to create a list of tasks and add them to a marketplace where other users can bid on completing those tasks. Lowest bid wins.

Friend Finder

Friendfinder is a full stack compatibility-based application thats takes in results from a user's survery and then compares their answers with those from other users. The app then displays the name and picture of the user with the best overall match.

travelBucket

travelBucket is an application that allows users to search travel destinations around the world and allows users to save a personalized list of their favorite places.

Hogwarts Train Scheduler

Train Scheduler, with a hogwarts twist, is an application that incorporates Firebase to host arrival and departure data inputed by a user. It retrieves and manipulates this stored information with Moment.js.

Eat-Da-Burger

Eat-Da-Burger is a full stack restaurant application created using Node, Express, Handlebars, MySQL and a custom ORM. It uses the MVC design pattern, Node and MySQL to query and route data to the app and Handlebars to generate the HTML.
